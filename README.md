
# BondBase

A simple web app to help you stay connected with your friends and manage your relationships easily.


## Tech Stack

- Next.js
- Tailwind CSS
- DaisyUI
- Recharts
- React-Toastify


## Features

- See friendship status: On-Track, Almost Due, or Overdue

- View friendship data and stats with simple charts

- Detailed profiles for every friend with their bio

- Responsive design that works on mobile, tablet and desktop
